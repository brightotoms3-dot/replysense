# **App Name**: ReplySense

## Core Features:

- Text/Screenshot Input: Allows users to input text or upload screenshots of messages or situations they need help with.
- Category Selection: Enables users to select a category (Relationship, Work/Boss, School, Family, Friends) to contextualize their request.
- AI Reply Generation: Generates three reply options (polite, confident, neutral) using AI, tailored to the input and category.
- Explanation Tool: Provides a short explanation of why each suggested reply works, using an AI reasoning tool.
- Reply Management: Allows users to copy suggested replies, regenerate new ones, and share the results.
- Usage Limits: Implements a limit of 3 reply generations per day for free users, with a soft upgrade message displayed after the limit is reached.
- Disclaimer Display: Shows a disclaimer stating 'AI suggestions only. Use your judgment.'

## Style Guidelines:

- Primary color: Deep indigo (#3F51B5) to create a calm, intelligent, and trustworthy atmosphere.
- Background color: Light gray (#F0F2F5), subtly desaturated from the primary to keep the focus on the content and give a clean feel.
- Accent color: Violet (#9C27B0), providing a pop of creative flair.
- Body and headline font: 'Inter', sans-serif, is used throughout for its modern, readable style. Note: currently only Google Fonts are supported.
- Use simple, outline-style icons for categories and actions, ensuring they are easily recognizable.
- Design a mobile-first layout with rounded cards to present reply options and explanations, maximizing readability and screen space.
- Incorporate subtle animations for loading states and transitions between screens to enhance user experience.
